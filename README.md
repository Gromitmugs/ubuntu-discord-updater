This is the auto-update script for your Discord application. Developed and tested on Ubuntu 22.04.

## Usage

- For the first time, runs:
`chmod +x script.sh`

- To run the script:
`./script.sh`

## Required Packages
- `dpkg` -> `sudo apt install dpkg`
- `curl` -> `sudo apt install curl`
